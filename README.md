# Deus.iA — Support site (GitHub Pages)

Static support page for App Store Connect **Support URL** (Guideline 1.5).

## Publish

1. Create a **public** repository on GitHub and push this folder as the repo root.
2. **Settings → Pages:** Source **Deploy from a branch**, branch `main`, folder **`/ (root)`**.
3. After deploy, set App Store Connect **Support URL** to `https://<user>.github.io/<repo>/`.

## Edit

- Update contact email and FAQ copy in `index.html` before going live.
